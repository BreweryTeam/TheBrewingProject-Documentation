Aging takes a long while, a typical beer could take about an hour to age, which is 3 in game days. You age a brew by putting it into a barrel, the type of the barrel matters.
Some brews age better in an oak barrel than a spruce barrel, and so on.

## Creating a barrel
You create a barrel by placing down all required blocks, and then lastly a sign facing towards any of these blocs. The sign can be removed after that without any issue.

If your server has not configured how the barrels will like, there should be two separate barrel types.
